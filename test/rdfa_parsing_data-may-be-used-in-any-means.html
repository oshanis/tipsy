This is an example webpage with RDFa embedded within it.

On page load the Emancy application will parse the metadata. That is it will store the data about the author, the content provider, the payment method etc.
This process will also work on any other webpage with RDFa embedded in an element with an id called "tipsy".

This particular test webpage also contains data which states that the user's data my be used in any way (stored/passed to third party etc)
If the user has configured his Emancy settings such that the terms of the vendor conflict with the user’s then the user will be informed.

Tips:
To see the RDFa embedded in this page right-click and select "View Source".
To see what has been stored as a result of visiting this test page click the small Emancy icon (bottom right) and select the "Activity" tab in the preference window that opens.
